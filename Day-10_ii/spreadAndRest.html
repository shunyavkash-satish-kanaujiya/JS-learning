Spread and Rest Operator

The spread and rest operators in JavaScript are used to manipulate arrays and objects.

Spread Operator (...)

The spread operator allows an iterable (like an array or string) to be expanded in places where multiple arguments or elements are expected.

Here are some examples of the spread operator:

Example 1: Concatenating arrays

Using the spread operator to concatenate two arrays:

        const arr1 = [1, 2, 3];
        const arr2 = [4, 5, 6];
        const combinedArray = [...arr1,...arr2];
        console.log(combinedArray); // Output: [1, 2, 3, 4, 5, 6]

Example 2: Merging objects

Using the spread operator to merge two objects:

        const obj1 = { a: 1, b: 2 };
        const obj2 = { c: 3, d: 4 };
        const mergedObject = {...obj1,...obj2 };
        console.log(mergedObject); // Output: { a: 1, b: 2, c: 3, d: 4 }

Rest Operator (...)

The rest operator allows an indefinite number of arguments to be unpacked into an array or an object.

Here are example of the rest operator:

Example :

        const numbers2 = [1, 2, 3, 4, 5];
        const [first, second, ...rest] = numbers2;
        console.log(first, second, rest); // Output: 1 2 [3, 4, 5]
