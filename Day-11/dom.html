DOM (Document Object Model)

DOM is a programming interface for HTML and XML documents. It provides a structured representation of the document's content and allows you to interact with it programmatically.

DOM allows you to manipulate the HTML structure, style, and content of a web page, as well as create dynamic and interactive web applications.

Benefits of DOM

  • Access and manipulate HTML elements dynamically
  • Create interactive web applications
  • Make web pages more responsive and efficient

Accesing the DOM in JS

There are several ways to access the DOM in JavaScript:

  • document.getElementById() :- finds an element by its id
  • document.getElementsByClassName() :- finds elements by their class name
  • document.getElementsByTagName() :- finds elements by their tag name
  • document.querySelector() and document.querySelectorAll() :- finds the first matching element or all matching elements

Modifying The Dom

There are several ways to modify the DOM in JavaScript:

  • innerHTML :- sets or gets the HTML content of an element
  • textContent :- sets or gets the text content of an element
  • setAttribute() and getAttribute() :- sets or gets attributes of an element

Creating and Adding element

There are several ways to create and add elements to the DOM in JavaScript:

  • createElement() :- creates a new element
  • appendChild() and insertBefore() :- adds or inserts an element as a child of another element
  • remove() :- removes an element from its parent
