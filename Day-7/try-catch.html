Try-Catch

The code in the try block is executed first, and if it throws an exception(error), the code in the catch block will be executed.

  • try-catch works synchronously
  • if you have any async function, script will die and catch won't work

Finally

  • if it exists it runs in all cases
  • of there is a return in the try or catch, it will still run
