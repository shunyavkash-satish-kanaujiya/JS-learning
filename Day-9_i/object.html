Object

Object is a non-primitive data type in JavaScript. It is a collection of properties and methods.

Objects are created using curly braces {} and have properties and methods defined inside them.

Here is an example of an object:

        let person = {
            name: "satish",
            age: 22,
            greet: function() {
                console.log('Hello, my name is'+ this.name);
            }
        };

To access a property or method of an object, you can use dot notation or square bracket notation.

For example, person.name will return 'satish', and person.greet() will log 'Hello, my name is satish'.

Objects can also have nested objects and arrays as properties.

Here is an example of an object with nested objects and arrays:

            let person = {
                name: "satish",
                age: 22,
                address: {
                    street: '123 Main St',
                    city: 'surat'
                },
                hobbies: ['reading', 'painting', 'cooking']
            };

To access a property or method of a nested object or array, you can use dot notation or square bracket notation with the nested property or array index.

For example, person.address.street will return '123 Main St', and person.hobbies[1] will return 'painting'.
