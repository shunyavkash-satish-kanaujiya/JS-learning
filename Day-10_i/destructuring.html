Destructuring

Destructuring is a JavaScript feature that allows you to unpack values from arrays or objects into distinct variables.

Here are some examples of destructuring in JavaScript:

Array Destructuring

        
        const numbers = [1, 2, 3, 4, 5];
        const [a, b, c, d, e] = numbers;
        console.log(a, b, c, d, e); // Output: 1 2 3 4 5

Object Destructuring

        
        const person = { name: 'sk', age: 22, occupation: 'Developer' };
        const { name, age, occupation } = person;
        console.log(name, age, occupation); // Output: sk 22 Developer

Nested Destructuring

        
        const person = { name: 'sk', age: 22, occupation: 'Developer', address: { street: '123 Main St', city: 'New York', state: 'NY' } };
        const { name, age, occupation, address: { street, city, state } } = person;
        console.log(name, age, occupation, street, city, state); // Output: sk 22 Developer 123 Main St New York NY

Default Values

        
        const person = { name: 'sk', age: 22 };
        const { name, age, occupation = 'Unknown' } = person;
        console.log(name, age, occupation); // Output: sk 22 Unknown
