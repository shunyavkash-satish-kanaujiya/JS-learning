Modules

JavaScript modules are a way to organize and structure your code. They allow you to create reusable pieces of code that can be imported and used in other parts of your application.

Types of Modules

  • ES6 Modules: These are the default type of module in JavaScript. They are defined using the `import` and `export` keywords.
  • you can export perticular function or variable using export keyword
    •     // myModule.js
    export const PI = 3.14;
    export function calculateArea(radius) {
        return PI * radius * radius;
    }
  • CommonJS Modules: These are also supported in Node.js. They are defined using `require()` and `module.exports`.
  • export default used to export a single value, function, or class from a module so that it can be imported without using curly braces {} in another file.
    •     // myModule.js
    export default function greet(name) {
        console.log(`Hello, ${name}!`);
    }
  • You can rename imports using 
    •     import { add as sum } from './path';    
    console.log(sum(4, 2)); // Output: 6
