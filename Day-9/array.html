Array

An array is a data structure in JavaScript that stores multiple values in a single variable.

Arrays can store values of different data types, such as numbers, strings, objects, and even other arrays.

Arrays are represented by square brackets ([]), and their elements are separated by commas.

Here are some examples of arrays:

  • const numbers = [1, 2, 3, 4, 5];
  • const fruits = ['apple', 'banana', 'orange'];
  • const mixedArray = [1, 'apple', { name: 'John' }, [4, 5, 6]];

Arrays have various methods and properties available for manipulation, such as push(), pop(), shift(), unshift(), splice(), slice(), and concat().

To access an element in an array, you can use its index, which starts from 0. For example, numbers[0] will return 1.

To modify an element in an array, you can assign a new value to its index, for example, numbers[0] = 0.
