Template Literals

Template literals are a feature in JavaScript that allows you to embed expressions inside strings. They are enclosed by backticks ( `....` ) and can contain placeholders denoted by ${expression}.

        
Example

        let name = "Satish Kanaujiya";
        let age = 22;
        let message = `Hello, my name is ${name}. I am ${age} years old.`;
        console.log(message); // Output: Hello, my name is Satish Kanaujiya. I am 22 years old.

Advantages

  • Improved readability
  • Easier string formatting
  • Support for multi-line strings
